GETTING STARTED WITH A NEW PROJECT
The project manager creates a new project by clicking on "Create New Project". The manager will receive two unique PIN codes. One PIN code for the intervention team and one code for the analysis team. The project manager distributes the appropriate PIN code to the project members depending on the specific role (interventionist or analyst). Members can sign in as either analyst or interventionist using the unique PIN. Members can only take one role per project
SPECIFY CRITERIA
The applications allows to specify up to 7 participants. Multiple members can be part of the intervention team and the analysis team. In this screen, the research criteria that both teams agree upon are specified.
Team members sign in as interventionist or analyst. You can only be assigned one role per project.
You will be able to navigate all your assigned or created research projects under two different tabs; Analyst and Interventionist.
The intervention team collects data. The interventionist immediately plugs in the data by clicking on the specific participant.
The members of the analysis team will be able to see the graphical display of the data in real time. Everytime the intervention team collects new data and plugs it in, the analysis team will see immediately the updated graphs.
Once the criteria specified in the beginning are met, the analysis team will receive a notification. The analysts will be directed to a decision screen. The analyst will need to make a decision (starting the intervention for another participant or further data collection is needed)
If the analysis team requests more data, a short explanation needs to be given.
In case there are multiple members in the analysis team and there is a disagreement (intervening versus more data collection), the analysts will be directed towards a new screen displaying an overview of the decisions made by all the analysis team members.
Before making a new decision, the analysts will discuss the decision and solve the discrapancy.
After the data collection is completed for all the participants, all the members of the analysis team will be guided to a final decision screen. The analysts will guess the order in which the interventions were introduced to the participants.
